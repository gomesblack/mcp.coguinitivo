# 12 — Critérios de Fechamento

## Objetivo
Definir quando uma fase ou camada pode realmente ser encerrada.

## Fechamento de fase
Uma fase só fecha se:
1. o bloco esperado foi produzido
2. o sistema registrou valor agregado
3. a pendência principal foi atualizada
4. o próximo papel foi definido ou a camada foi dada como concluída
5. o menu seguinte foi gerado

## Fechamento de camada
Uma camada só fecha se:
1. todas as fases foram percorridas
2. não há pendência crítica aberta
3. o envelope foi gerado
4. a próxima camada foi formalmente liberada

## Fechamento da reunião
A reunião só fecha se:
1. a Camada 3 estiver encerrada
2. o pacote operacional estiver completo
3. o modo X estiver pronto para abertura
